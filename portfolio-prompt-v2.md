## 파일 구성

작업 폴더 안에 아래처럼 준비되어 있다:

```
/my-portfolio
  ├── index.html        ← 이 파일을 만든다
  └── fonts/
        ├── Paperlogy-8ExtraBold.ttf
        └── Paperlogy-4Regular.ttf
```

폰트 파일은 같은 폴더 내 `fonts/` 디렉토리에 있다.
Google Fonts나 CDN은 사용하지 않는다. 로컬 폰트만 사용한다.

## 디자인 시스템

### 폰트 설정

```css
@font-face {
  font-family: 'Paperlogy';
  src: url('fonts/Paperlogy-8ExtraBold.ttf') format('truetype');
  font-weight: 800;
  font-style: normal;
}

@font-face {
  font-family: 'Paperlogy';
  src: url('fonts/Paperlogy-4Regular.ttf') format('truetype');
  font-weight: 400;
  font-style: normal;
}

:root {
  --font-display: 'Paperlogy', sans-serif;
  --font-body: 'Paperlogy', sans-serif;
}
```

- 대제목(Hero, 섹션 타이틀): `font-weight: 800`
- 본문, 레이블, 네비: `font-weight: 400`
- 중간 강조가 필요하면 800과 400 사이를 CSS로 조정하지 말고 400으로 통일한다 (이 폰트는 두 weight만 존재)

### 컬러 시스템

```css
:root {
  --color-bg:        #ffffff;
  --color-bg-soft:   #f5f5f5;
  --color-bg-muted:  #efefef;
  --color-text:      #111111;
  --color-text-sub:  #555555;
  --color-text-hint: #999999;
  --color-border:    #e0e0e0;
  --color-accent:    #cb4652;
  --color-accent-bg: rgba(203, 70, 82, 0.07);
}
```

**컬러 원칙:**
- 배경: 흰색 (`#ffffff`) 기반. 구분이 필요한 섹션은 `#f5f5f5` 사용
- 텍스트: `#111111` (주요), `#555555` (보조), `#999999` (힌트)
- 포인트 컬러 `#cb4652` 는 딱 하나의 용도로만 사용: **섹션 라벨, 강조 뱃지, 호버 언더라인, 활성 상태 표시**
- 버튼 배경에도 `#cb4652` 사용 가능하나 남발 금지
- 섀도 사용 금지 — 구분은 border와 배경색 차이로만
- 그라디언트 사용 금지

|------|------|------|

### 형태

```css

  --radius-pill:    32px;
  --radius-pill-lg: 40px;
  --radius-card:    0px;
  --radius-sm:      4px;
}
```

- 버튼: 항상 pill (32px ~ 40px)
- 카드: 0px (날카로운 모서리)
- 섹션 컨테이너: 4px 또는 0px

## 배포 방법 (완성 후)

### 방법 A — Netlify Drop (가장 빠름, 폰트 파일 포함 가능)
```
1. https://app.netlify.com/drop 접속
2. my-portfolio 폴더 전체를 브라우저에 드래그 앤 드롭
   (index.html + fonts/ 폴더가 함께 올라가야 폰트가 정상 작동)
3. 즉시 URL 발급
```

### 방법 B — GitHub Pages
```bash
git init
git add .
git commit -m "Add portfolio"
git remote add origin https://github.com/{username}/{repo}.git
git push -u origin main
# Settings → Pages → Branch: main / root → Save
```

### 방법 C — Netlify CLI
```bash
npm install -g netlify-cli
netlify login
netlify deploy --dir=. --prod
```
