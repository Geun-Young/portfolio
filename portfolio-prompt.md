## 디자인 시스템 (Mercury Design System)

아래 CSS 변수를 반드시 사용한다.

```css
:root {
  /* Colors */
  --color-mercury-blue: #5266eb;
  --color-ghost-blue: #cdddff;
  --color-deep-space: #171721;
  --color-midnight-slate: #1e1e2a;
  --color-graphite: #272735;
  --color-lead: #70707d;
  --color-starlight: #ededf3;
  --color-silver: #c3c3cc;
  --color-pure-white: #ffffff;

  /* Fonts — Google Fonts Manrope로 대체 */
  --font-display: 'Manrope', sans-serif;
  --font-body: 'Manrope', sans-serif;

  /* Radius */
  --radius-pill: 32px;
  --radius-pill-lg: 40px;
  --radius-card: 0px;
  --radius-sm: 4px;
}
```

**디자인 원칙:**
- 배경: `#171721` (deep-space) / `#1e1e2a` (midnight-slate)
- 주요 텍스트: `#ededf3` (starlight)
- 보조 텍스트: `#c3c3cc` (silver)
- 포인트 컬러: `#5266eb` (mercury-blue) — CTA 버튼에만 단독 사용
- 버튼은 항상 pill 형태 (border-radius: 32px 또는 40px)
- 카드의 border-radius: 0px
- 섀도 사용 금지 — 호버 시 색상/투명도 변화로 elevation 표현
- 섹션 간격: 80px ~ 120px
- 폰트 굵기: 300(light), 400, 500 — 600 이상 사용 금지
- 큰 제목은 font-weight: 300 (light)으로 권위감 표현

## 구현 요구사항

### 파일 구성
- `index.html` 단일 파일 (CSS, JS 모두 인라인)
- Google Fonts: `Manrope` weight 300, 400, 500 로드

### Nav
- `position: fixed` 상단 고정
- 배경: `rgba(23, 23, 33, 0.75)` + `backdrop-filter: blur(20px)`
- 좌: 이름 로고 (Manrope 500)
- 우: 경력 / 자격증 / 수상 / 프로젝트 링크 + "연락하기" ghost pill 버튼
- 앵커 링크로 각 섹션으로 스크롤

### Hero 섹션
- 전체 뷰포트 높이 (min-height: 100vh)
- 배경: `#171721` + 미묘한 파란 radial-gradient glow (opacity 낮게)
- 미세한 격자 패턴 배경 (::after pseudo-element)
- 중앙 정렬 레이아웃
- 애니메이션: 요소들이 순차적으로 fadeUp (animation-delay 스태거)
- 하단: 프로젝트 수, 자격증 수, 수상 수 stat 표시 (숫자 + 라벨)
- CTA 버튼 2개: "프로젝트 보기 →" (primary blue), "이메일 주소" (ghost)

### 경력/학력 섹션
- 2컬럼 그리드 (좌: 경력, 우: 학력)
- 타임라인 형태: 날짜 + 직책명 + 설명
- 아이템 하단에 1px border로 구분

### 자격증 섹션
- 그리드 레이아웃 (auto-fill, minmax 280px)
- 1px 구분선으로 된 그리드 테두리 (mosaic 스타일)
- 각 셀: 아이콘 원 + 자격증명 + 취득 연도
- 호버 시 배경 `#272735`로 변경

### 수상경력 섹션
- 리스트 형태
- 각 항목: 연도 | 뱃지 (mercury-blue ghost) | 수상명
- 호버 시 좌측으로 살짝 이동 (padding-left transition)
- 상하 1px border로 구분

### 프로젝트 섹션
- 아코디언 형태 (클릭하면 상세 내용 토글)
- 닫힌 상태: 번호 + 프로젝트명 + 태그들 + + 버튼
- 열린 상태: 배경 / 접근 / 결과 / 기여 4개 섹션 노출
- `max-height` 트랜지션으로 부드러운 애니메이션
- + 버튼은 열릴 때 ×로 회전 (rotate(45deg))

### 스킬 섹션 (선택적으로 삽입)
- 배경: `#1e1e2a` 전체 너비 배경
- 4개 카드 그리드 (SQL, Python, R, SAS)
- 각 카드: 스킬명 + 설명 + 가로 바 (레벨 시각화)
- 바는 CSS animation으로 로드 시 채워지는 효과

### Footer
- 큰 타이포: "함께 일해요" (Manrope 300, 65px)
- 연락처 링크 버튼들: 이메일, 전화번호
- 하단: copyright 텍스트

### 애니메이션
- 스크롤 감지 (IntersectionObserver)로 섹션 진입 시 `fadeUp` reveal
- Hero 요소들은 페이지 로드 시 순차 등장 (animation-delay 0.2s ~ 1s)
- 모든 transition은 0.2s ~ 0.4s ease

### 반응형
- 모바일 (768px 이하): 1컬럼, 패딩 24px, 네비 링크 숨김
- 태블릿 이상: 현재 레이아웃 유지
