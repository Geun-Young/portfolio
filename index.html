페이어텐션 구독자 활성도 분석 및 콘텐츠 전환 지표 재정의

노출편향 퍼널 지표 설계 전환율 분석
+

배경

구독 서비스의 활성도 기여 여부 불명확. 기존 CTR 지표로는 노출 편향으로 인해 실제 유저 관심도 측정 불가.

접근

구독 전후 Pre-Post 분석으로 활성도 인과 효과 검증. 도메인 메뉴 클릭 → 저널 클릭 → CTA 클릭의 3단계 퍼널 전환 지표로 재설계.

결과

구독자와 비구독자 대비 체류시간·방문빈도·이용 서비스 수 전 항목에서 약 2배 이상 차이 확인. 구독 이후 활성도 추가 증가 → 구독 효과가 행동 변화임을 검증.

기여

퍼널 전환 지표 재정의로 콘텐츠 메뉴 구성 개편을 위한 데이터 기반 의사결정 근거 제공.

+

배경

접근

포인트 테이블·결제 테이블 기준 지표 각각 산출 후 수치 비교. 특정 결제 건 약 10건 샘플링하여 단건 단위로 직접 검증 → 포인트 적립 N건이 1회 사용 시 N건으로 중복 산정되는 적재 오류 규명.

결과

사용자 ID·거래 시점 기준으로 포인트 사용 건을 단일 거래로 집계, 금액은 합산 기준으로 지표 산출 로직 개선. 결제팀·기획팀 간 정합성 확보.

기여

정정된 분석 결과를 HD현대그룹-카카오페이 MOU 자리에서 공식 활용. 이후 HD현대삼호·HD현대일렉트릭 등 계열사로 제휴 확대 적용.

+

배경

거리순 정렬 필터 도입 후 실제 결제 행동에 미치는 영향 검증 필요. 로그 데이터 상 거리순·추천순 노출 구분 정보 부재로 분석 기준 신규 정립 필요.

접근

거리순 필터 클릭 → 바코드 화면 진입 → 브랜드 일치 → 결제 완료 흐름으로 거리순 활용 유저 직접 정의. 기능 효과와 유저 특성 효과를 분리하기 위해 도입 전후 × 신규·기존 유저 4집단으로 설계.

결과

도입 후 신규 유저의 거리순 필터 이용률이 도입 전 대비 높게 나타남. 거리순 활용 신규 유저의 1인당 TPV가 미활용 대비 약 150% 높게 측정됨.

기여

거리순 정렬이 신규 유저의 탐색 부담을 완화하고 결제 전환을 촉진하는 기능임을 데이터로 검증.

이벤트 성과 분석 유저 세그먼트 정의 proxy 지표 설계
+

배경

이벤트 캠페인 성과 측정 시 신규·부활 유저를 정확히 집계하기 어려운 구조적 문제 존재. 다양한 진입 경로로 인해 이벤트 페이지 직접 진입 기준만으로는 실제 이벤트 유입 유저를 온전히 포착 불가.

접근

로그 데이터 제약을 우회하는 proxy 기준을 직접 설계. 신규 유저: 회원가입 후 10분 내 이벤트 페이지 방문. 부활 유저: 1개월 이상 미접속 후, 첫 방문으로부터 5분 내 이벤트 페이지 접속.

결과

설계한 유저 정의 로직이 카카오페이 퍼포먼스 마케팅팀의 이벤트 성과 측정 공식 기준으로 채택되어 현재까지 실무에 사용 중.

기여

데이터 추적 한계 상황에서 분석 기준 자체를 설계하고, 해당 정의가 퍼포먼스팀 공식 로직으로 이어지는 데 기여.
